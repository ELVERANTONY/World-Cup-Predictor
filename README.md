# World Cup Predictor 2026

Predictor de resultados del Mundial 2026 con autenticación, rankings, salas de predicción y más.

## Stack

- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: Node.js + Express + TypeScript + Prisma + PostgreSQL
- **Autenticación**: JWT + Google OAuth 2.0
- **Despliegue**: Frontend en Vercel, Backend en Render

## Estructura

```
backend-predicciones-mundial/   → API REST (Express + Prisma + PostgreSQL)
frontend-predicciones-mundial/  → SPA (React + Vite + Tailwind)
```
